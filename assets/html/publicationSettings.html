Publication Settings

unHack supports two publishing methods. Click one of the tabs to choose and enter your information.

unHack will store the Password on your Computer within it's App storage. The password isn't shared or saved to any file. If you want to change the stored password, just type the new one into the password field and unHack will updated its storage.

GitHub Pages will be used. Make sure to install Git and have SSH keys added to GitHub ( Tutorial). Go to Documentation for more information on how to correctly use.
